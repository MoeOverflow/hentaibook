# hentaibook
convert  comic on [nhentai.net](https://nhentai.net) to .epub e-book

## Install

``` Shell
$ pip install -r requirements.txt
```

## Usage

``` Shell
$ cd hentaibook
$ python3 main.py -l https://nhentai.net/g/123170/

$ python3 main.py --help
```
